# NitroBlack 🚗⚡

Sistema web completo de locadora de veículos premium desenvolvido com foco em experiência visual moderna, gerenciamento administrativo e reservas online.

---

## 📌 Sobre o projeto

A **NitroBlack** é uma plataforma de aluguel de carros premium com visual inspirado em interfaces cyberpunk e foco em performance, organização e experiência do usuário.

O sistema permite:

- Visualização de veículos disponíveis
- Reserva de carros
- Carrinho de reservas
- Painel administrativo
- Cadastro e gerenciamento de veículos
- Cadastro de usuários
- Sistema de login
- Controle de administradores
- Integração com banco de dados

---
